R Programming for Statistical & Genomic Data Analysis

1 Day-1 : Fundamentals of R Programming

1.1 Learning Objectives

By the end of this day, students will be able to:

  • Understand what R is and where it is used
  • Use R as a calculator
  • Create and manipulate basic R objects
  • Work with vectors and data frames
  • Use basic R functions and help documentation

1.2 What Is R?

R is a programming language and environment designed for:

  • Statistical analysis
  • Data visualization
  • Reproducible research

R is widely used in: - Statistics - Epidemiology - Genomics - Social sciences - Machine learning research

One of R‚Äôs biggest strengths is its ecosystem of packages, including
CRAN and Bioconductor.

RStudio provides a user-friendly interface for working with R.

Main components: - Console ‚Äì where commands are executed - Script Editor ‚Äì where you write and save code - Environment ‚Äì shows created objects - Plots / Files / Help ‚Äì outputs and documentation

üí° Tip:

1.5 Creating Objects in R

R stores values in objects.

Assignment is typically done using <-:

x <- 10
y <- 3

Use the objects in calculations:

x + y
## [1] 13
x * y
## [1] 30

üí° Tip The ‚Äú=‚Äù operator also works, but <- is the recommended R convention.

1.7 Data Structures

1.7.1 Vector

A vector is a collection of elements of the same data type.

numbers <- c(1, 2, 3, 4, 5)
genes <- c("TP53", "BRCA1", "EGFR")
flags <- c(TRUE, FALSE, TRUE)

Indexing vectors:

numbers[1]
## [1] 1
numbers[2:4]
## [1] 2 3 4

1.8 Vectorization: Thinking in Vectors

One of the key ideas in R is vectorization.

In R, most operations are designed to work on entire vectors at once, rather than one element at a time. This makes R code:

  • Simpler to write
  • Easier to read
  • Faster to execute

This is different from many other programming languages, where you often need loops to process each value individually.

numbers * 2
## [1]  2  4  6  8 10
numbers + 10
## [1] 11 12 13 14 15

1.8.1 Data Frames

A data frame is a table-like structure where each column can be a different data type.

df <- data.frame(
id = 1:5,
age = c(23, 35, 41, 29, 50),
group = c("A", "B", "A", "B", "A")
)

df
##   id age group
## 1  1  23     A
## 2  2  35     B
## 3  3  41     A
## 4  4  29     B
## 5  5  50     A

Inspecting a data frame:
